Orange Money

Procédure USSD et étapes détaillées

1
Tapez *144# sur votre téléphone puis validez (OK / YES).
2
Choisissez la devise 2 - CDF OK .
3
Sélectionnez l'option : 3 OK.
4
Sélectionnez l'option Retrait agent : 1 OK.
5
Entrez le numéro de l'agent : 0858009376.
5
Indiquez le montant dépôt (ex : 50 000 CDF).
6
Confirmez en saisissant votre code secret Orange Money.
7
Recevez la confirmation SMS.
INFO :

Après avoir effectué votre transaction via l’opérateur de votre choix, cliquez sur le bouton « Dépôt ». Un formulaire s’affichera et vous sera demandé de :
Choisir la méthode de paiement que vous avez utilisés: -. Orange Money, M-Pesa ou Airtel Money.
- Saisir le montant déposé. Une fois validé, vous serez redirigé vers un champ où vous devrez renseigner la référence de transaction. Cette référence vous est envoyée par SMS après votre opération. 👉 Exemple de référence ( ref: CO250918.0009.A39300 ) Veuillez copier et coller ce code exactement tel qu’indiqué dans votre SMS afin que votre dépôt soit correctement validé et pris en compte.

Vodacom — M-Pesa

Procédure USSD et confirmation

1
Tapez *1122# puis validez.
2
Choisissez la devise 2 - FC OK .
3
Sélectionnez l'option : 3 OK.
4
Sélectionnez l'option : 1 - Agent OK.
5
Entrez le numéro de l'agent M-Pesa : pas disponible.
6
Indiquez le montant que vous souhaitez deposer.
5
Confirmez la transaction avec votre code PIN M-Pesa.
6
Vous recevrez un SMS de confirmation .
INFO :

Après avoir effectué votre transaction via l’opérateur de votre choix, cliquez sur le bouton « Dépôt ». Un formulaire s’affichera et vous sera demandé de :

- Saisir le montant déposé. Une fois validé, vous serez redirigé vers un champ où vous devrez renseigner la référence de transaction. Cette référence vous est envoyée par SMS après votre opération. 👉 Exemple de référence ( ref: PP542.7257628.78 ) Veuillez copier et coller ce code exactement tel qu’indiqué dans votre SMS afin que votre dépôt soit correctement validé et pris en compte.

Airtel Money

Code USSD & étapes

1
Tapez *501# et validez.
2
Choisissez la devise 2 - CDF OK .
3
Sélectionnez option: 1 OK.
3
Sélectionnez option (Vers Aitel Money): 1 OK.
4
Sélectionnez option (avec frais): 2 OK.
5
Indiquez le numéro de l'agent : pas disponible.
6
Saisissez le montant.
7
Confirmez avec votre code PIN Airtel Money.
8
Montrez le SMS de confirmation à l'agent .
INFO :

Après avoir effectué votre transaction via l’opérateur de votre choix, cliquez sur le bouton « Dépôt ». Un formulaire s’affichera et vous sera demandé de :
Choisir la méthode de paiement que vous avez utilisés: -. Orange Money, M-Pesa ou Airtel Money.
- Saisir le montant déposé. Une fois validé, vous serez redirigé vers un champ où vous devrez renseigner la référence de transaction. Cette référence vous est envoyée par SMS après votre opération. 👉 Exemple de référence ( ref: PP542.7257628.78 ) Veuillez copier et coller ce code exactement tel qu’indiqué dans votre SMS afin que votre dépôt soit correctement validé et pris en compte.
